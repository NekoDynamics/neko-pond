# Experiments

## 中文

`Experiments` 存放短周期验证、交互原型、性能测试和行为观察实验。实验代码不得直接进入主路径，除非经过性能与架构复审。

## English

`Experiments` contains short-cycle validation, interaction prototypes, performance tests, and behavior observation experiments. Experimental code must not enter the main path without performance and architecture review.
