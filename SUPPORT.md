# 支持 / Support

## 中文

### 获取帮助

- 阅读根目录 README 与 `Docs/zh-CN`
- 检查现有 issue 与 milestone
- 优先报告可复现的行为与性能问题

### 适合提交的问题

- 触摸响应异常
- 帧率或延迟回退
- iPad 兼容性问题
- AI 行为不稳定
- 渲染结果与预期不符

### 不适合提交的问题

- 没有设备信息的问题
- 没有复现步骤的问题
- 纯主观审美意见，没有环境说明的问题

## English

### Getting Help

- Read the root README and `Docs/en-US`
- Check existing issues and milestones
- Prefer reproducible behavior and performance reports

### Good Reports

- Touch response anomalies
- Frame-rate or latency regressions
- iPad compatibility problems
- Unstable AI behavior
- Rendering results that differ from expectations

### Poor Reports

- Issues without device information
- Issues without reproduction steps
- Pure aesthetic feedback with no environment context
