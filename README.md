# Neko Pond

Not a game for humans.
A living digital habitat for cats.

A next-generation interactive ecosystem for cats on iPadOS.

Built with:
- SwiftUI
- SpriteKit
- Metal
- CoreML

Designed for:
- ultra-low latency touch interaction
- natural prey simulation
- AI-driven behavioral adaptation
- immersive ambient environments

## Positioning

Neko Pond is not a traditional mobile game.

It is a real-time interactive ecosystem engineered specifically for feline perception, movement patterns, and hunting instincts.

## MVP

- AI Fish Pond
- Organic fish movement
- Dynamic water ripple simulation
- Cat paw interaction system
- Ambient sound feedback
- Guided Access support
- Multi-touch filtering
- Ultra-low latency rendering

## Product Direction

An AI-powered interactive ecosystem designed specifically for cats.

App Store category: Entertainment.

Platform focus: iPadOS first.
