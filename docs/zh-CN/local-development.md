# 本地开发

## 中文

### 目标

本仓库的本地开发应当围绕稳定的 iPadOS 交互体验、可验证的渲染路径和可控的 AI 行为展开。

### 推荐流程

1. 阅读 `README.md`
2. 检查 `App`、`Engine`、`AI`、`Tools`
3. 先验证渲染，再验证触摸，再验证行为
4. 最后检查性能预算

## English

### Goal

Local development should stay centered on stable iPadOS interaction, verifiable rendering paths, and controllable AI behavior.

### Recommended Flow

1. Read `README.md`
2. Inspect `App`, `Engine`, `AI`, `Tools`
3. Validate rendering first, touch second, behavior third
4. Finish with performance budgets
