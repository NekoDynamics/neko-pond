# Product Architecture

## One-Line Positioning

An AI-powered interactive ecosystem designed specifically for cats.

## Core Experience

Neko Pond creates a living digital pond on iPadOS where cats can observe, track, and interact with AI-driven aquatic motion.

## MVP Scope

- AI Fish Pond
- Organic fish movement
- Dynamic water ripple simulation
- Cat paw interaction system
- Ambient sound feedback
- Guided Access support
- Multi-touch filtering
- Ultra-low latency rendering

## Technical Direction

- SwiftUI for app shell and settings
- SpriteKit for interactive scene composition
- Metal for high-performance visual effects
- CoreML for behavioral adaptation

## Interaction Principles

- Prioritize feline perception over human game loops.
- Keep latency low enough for paw-driven interaction.
- Favor organic motion over predictable paths.
- Use high-contrast visuals suitable for dark ambient environments.
- Design for supervised indoor use on iPadOS.

## Commercial Direction

- Platform: iPadOS only for the first stage
- Model: Freemium plus subscription
- Category: Entertainment
