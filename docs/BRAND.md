# Brand

## Naming

- GitHub repository: `neko-pond`
- App name: Neko Pond
- Product line: Neko
- Positioning: AI Cat Ecosystem

## App Store

- Subtitle: A Living Playground for Indoor Cats
- Category: Entertainment

## Tagline

Not a game for humans.
A living digital habitat for cats.

## Voice

Use:
- Ambient
- Natural
- AI-driven
- Organic Motion
- Interactive Ecosystem
- Designed for Cats

Avoid:
- fun
- cute
- mini game

## Visual Direction

Dark Ambient.

Preferred logo directions:
- minimal water ripple
- abstract fish form
- dynamic ring
- black background with luminous fish

Avoid:
- cartoon cat head
- paw print
- low-end pet UI
- mascot-first branding
