# Ecosystems

## 中文

生态模块承载产品级互动逻辑，例如 Neko Pond、Neko Swarm、Neko Sky 和 Neko Insight 的场景规则。共享能力应下沉到 `Engine` 或 `AI`，避免单一生态污染核心层。

## English

Ecosystem modules contain product-level interaction logic for Neko Pond, Neko Swarm, Neko Sky, and Neko Insight. Shared capabilities should move into `Engine` or `AI` so one ecosystem does not pollute the core layer.
